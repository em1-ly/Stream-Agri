# App Store Distribution Setup

## Prerequisites
1. Apple Developer Account ($99/year) - You already have team ID: Q8SVVQWR37
2. Expo account (free)

## Step 1: Login to EAS
Run in terminal:
```bash
cd /Users/curveridsupport/Stream-Agri
eas login
```

## Step 2: Configure Apple Credentials
EAS can automatically manage certificates and provisioning profiles:
```bash
eas build:configure
```

Or manually configure:
```bash
eas credentials
```

## Step 3: Build for App Store
```bash
eas build --platform ios --profile production
```

This will:
- Automatically exclude dev dependencies (expo-dev-menu)
- Generate/use App Store certificates
- Create provisioning profiles
- Build the IPA
- Upload to EAS servers for download

## Step 4: Download and Submit
1. Download the IPA from the EAS build page
2. Or use: `eas submit --platform ios --latest`

## Alternative: Local Build (if you prefer)
If you want to build locally in Xcode:
1. Open Xcode: `open ios/TobaccoLogistics.xcworkspace`
2. Select "Any iOS Device" or "Generic iOS Device"
3. Product > Archive
4. Window > Organizer > Distribute App > App Store Connect
